# Haziq Ariff Portfolio

This is a modern, production-quality portfolio built with Next.js, TypeScript, and Framer Motion.

## Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS 4+
- **Components**: shadcn/ui
- **Animation**: Framer Motion
- **Icons**: react-icons
- **Theming**: next-themes
- **Validation**: React Hook Form + Zod

## Features

- [ ] Responsive Home, About, Skills, Projects, and Contact pages.
- [ ] Glassmorphism design language.
- [ ] Dark/Light mode support.
- [ ] High-fidelity entry animations.

## Development

```bash
npm run dev
```
